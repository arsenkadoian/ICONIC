1. Before you run the programe, in your command type the following:

-> conda activate base -> conda install -c conda-forge notebook nb_conda_kernels -> conda create --name NEW_ENV python=3.9 ipykernel -> conda activate NEW_ENV

2. Now position yourself in the same folder where you place this folder and type the following

-> pip install -r requirements.txt

3. To initialize it, type:

-> python HandIn.py --country1 <"a country of your choice">, --country2 <"a country of your choice">, 
                    --datapoint <"a datapoint of your choice other than 'date'">,

4. "ENJOY"!
